# CampusMonopoly 校园版大富翁改进规则

## 一、游戏定位

**游戏目标**：通过本地轮流回合制对局，改善大学生宿舍交流弱、储蓄思维不足、校园社交渠道不足的问题。

**核心设计方向**：

1. 让玩家在对局中主动沟通、协商、互助，而不是只进行个人资金竞争。
2. 让储蓄、消费、兼职和风险管理形成可感知的长期收益。
3. 让校园场景承载社交事件，使玩家在游戏中自然产生宿舍互动和校园连接。

## 二、基础对局规则

### 1. 参与人数

2-4 人，本地轮流回合制游玩。开局时创建“宿舍专属房间”，玩家以同一宿舍成员身份进入游戏。

### 2. 初始资金

玩家自定义每月生活费，金额区间为 1000-3000 元。所有玩家初始资金一致。

推荐设置：

- 轻松局：3000 元
- 标准局：2000 元
- 生存局：1000 元

### 3. 地图地块

地图包含以下校园专属地块：

- 教学楼
- 食堂
- 图书馆
- 超市
- 快递站
- 奶茶店
- 外卖店
- 网购点
- 电影院
- 兼职中心
- 宿舍
- 银行
- 考点
- 社团广场
- 心愿墙
- 宿舍议事厅

新增社交类地块用于增强宿舍沟通和校园社交：

- **社团广场**：可邀请 1 名玩家进行社交互动，双方各获得 1 点社交值。
- **心愿墙**：玩家可公开一个本轮小目标，例如“存到 2500 元”“本轮不消费”“帮助室友一次”，完成后获得奖励。
- **宿舍议事厅**：所有未出局玩家可进行一次 60 秒协商，讨论是否共同承担费用、组织团建或建立储蓄计划。

### 4. 回合流程

每名玩家的回合按以下顺序执行：

1. 投掷骰子。
2. 移动对应步数。
3. 结算途经或停留地块的固定消费。
4. 触发落地地块事件。
5. 随机触发 1 个被动事件。
6. 抽取 1 张行动牌。
7. 可选择使用 1 张行动牌。
8. 可执行 1 次地块允许的主动操作，例如兼职、存款、学习、休息、社交互动。
9. 回合结束，结算余额宝收益、定期存款到期、心情值与精力值恢复、状态倒计时。
10. 下一名玩家接续回合。

### 5. 手牌规则

每回合固定抽取 1 张行动牌，手牌上限为 3 张。若抽牌后超过上限，必须弃掉 1 张。

每回合最多使用 1 张行动牌，使用后立即作废。

### 6. 胜利条件

默认胜利条件：最后剩余 1 名未破产、未出局的玩家，即为本局获胜者。

推荐新增“宿舍共赢结局”：

当游戏进行满 20 回合后，若仍有至少 2 名玩家未破产，且所有存活玩家的平均储蓄率达到 25%，则触发宿舍共赢结局。此时资金最高者为“理财达人”，社交值最高者为“宿舍核心”，互助次数最高者为“靠谱室友”。

该结局用于避免游戏只鼓励互相伤害，让玩家在竞争之外也能形成合作目标。

### 7. 破产与出局

玩家资金小于或等于 0 时判定为破产，直接淘汰出局。

出局玩家不再行动，但可以参与旁观讨论，不再进行资金结算。

## 三、核心经济规则

### 1. 固定收入来源

#### 父母转账

每完整 4 回合，玩家自动获得初始生活费 20% 的金额。

#### 校园兼职

停留在兼职中心地块时，玩家可选择执行兼职，单次获得 100-300 元随机金额。

若玩家处于“考试挂科补考”影响期间，接下来 3 回合禁止执行兼职。

兼职会消耗 30 点精力值，并使心情值 -5。玩家不能连续两个自己的回合都执行兼职；若上一回合已经兼职，本回合必须选择其他行动。

若玩家精力值低于 30，则不能执行兼职。若玩家认知水平达到 60，兼职收益额外 +50 元；达到 80，兼职收益额外 +100 元。

#### 奖学金

停留在教学楼地块时，有概率触发奖学金发放，获得 300-500 元。

若玩家拥有“学习状态”标记，奖学金触发概率提高。若玩家成绩达到 80，奖学金触发概率额外提高；成绩达到 90 时，奖学金金额额外 +100 元。

#### 余额宝收益

回合结束时，玩家持有资金的 5% 计入活期收益并自动到账。

为防止滚雪球过强，单回合余额宝收益最高不超过 150 元。

### 2. 常规支出场景

途经或停留以下地块时，自动扣除对应消费金额：

| 地块 | 消费金额 |
| --- | ---: |
| 奶茶店 | 50 元 |
| 外卖店 | 80 元 |
| 网购点 | 150 元 |
| 电影院 | 120 元 |
| 食堂 | 40 元 |
| 超市 | 60 元 |

若玩家本回合拥有“校园消费券”或“拼单省钱”效果，则按对应规则减免。

### 3. 储蓄机制

#### 银行定期存款

停留在银行地块时，玩家可自愿存入资金。存入后 3 回合内无法支取，到期返还本金和 10% 利息。

每名玩家最多同时持有 2 笔定期存款。

#### 储蓄目标

每名玩家开局时可设置一个储蓄目标：

  | 储蓄目标 | 要求                 | 完成奖励                                  |
  | :------- | :------------------- | :---------------------------------------- |
  | 保守目标 | 存下初始生活费的 20% | 获得 1 张行动牌                           |
  | 标准目标 | 存下初始生活费的 35% | 获得 1 张行动牌 + 1 点理财值              |
  | 挑战目标 | 存下初始生活费的 50% | 获得 2 张行动牌 + 2 点理财值 + 100 元奖励 |

储蓄目标鼓励玩家思考消费延迟满足，而不是只依赖随机收入。

#### 宿舍共同基金

所有玩家可在宿舍议事厅协商建立宿舍共同基金。每名参与玩家一次性投入 100 元。

共同基金可用于：

- 抵扣一次所有玩家共同支出。
- 替一名濒临破产玩家支付最多 150 元。
- 在团建时抵扣总费用 200 元。

共同基金不属于任何单个玩家，不能被个人取出。

### 4. 特殊支出

#### 宿舍水电费

停留在宿舍地块时，每 2 回合自动扣除水电费 50 元。

若玩家所在宿舍共同基金余额充足，可由共同基金支付。

#### 考试报名费

停留在考点地块时，需扣除考试报名费 100 元。

若玩家拥有“学霸辅导”相关互动记录，可减免 50 元。

## 四、成长数值规则

除资金、社交值、理财值外，每名玩家新增 4 个成长数值：心情值、精力值、成绩、认知水平。

这些数值用于让玩家在兼职赚钱、学习提升、社交放松和储蓄规划之间形成取舍，避免游戏只围绕金钱增减。

### 1. 初始数值

| 数值 | 初始值 | 范围 | 含义 |
| --- | ---: | ---: | --- |
| 心情值 | 70 | 0-100 | 代表玩家的情绪状态，影响社交、消费冲动和部分负面事件。 |
| 精力值 | 80 | 0-100 | 代表玩家可用于学习、兼职和社交的行动能力。 |
| 成绩 | 60 | 0-100 | 代表学业表现，影响奖学金、考试费用和补考风险。 |
| 认知水平 | 50 | 0-100 | 代表理财、规划和校园信息理解能力，影响储蓄收益、兼职效率和事件判断。 |

### 2. 数值变化规则

| 行为或事件 | 心情值 | 精力值 | 成绩 | 认知水平 |
| --- | ---: | ---: | ---: | ---: |
| 执行兼职 | -5 | -30 | 0 | +3 |
| 在图书馆学习 | -5 | -20 | +8 | +5 |
| 在宿舍休息 | +15 | +25 | 0 | 0 |
| 参与团建或社交互动 | +10 | -10 | 0 | +2 |
| 制定预算 | 0 | 0 | 0 | +5 |
| 成功完成储蓄目标 | +5 | 0 | 0 | +8 |
| 考试挂科补考 | -15 | -10 | -8 | 0 |
| 获得奖学金 | +10 | 0 | +5 | +3 |
| 过度消费事件 | +5 | 0 | 0 | -3 |

### 3. 阈值效果

| 数值 | 阈值 | 效果 |
| --- | ---: | --- |
| 心情值 | 低于 30 | 进入“低落”状态，不能主动发起团建，抽到消费类事件时额外扣除 50 元冲动消费。 |
| 心情值 | 达到 85 | 进入“心情良好”状态，下一次社交互动额外获得 1 点社交值。 |
| 精力值 | 低于 30 | 不能兼职，不能在图书馆学习，只能进行休息、储蓄、普通移动和低消耗社交。 |
| 精力值 | 达到 90 | 本回合学习或兼职收益额外 +20%。 |
| 成绩 | 低于 50 | 考点地块有更高概率触发补考事件。 |
| 成绩 | 达到 80 | 奖学金触发概率提高，考试报名费减免 50 元。 |
| 认知水平 | 低于 40 | 银行定期存款收益保持 10%，但不能获得额外理财奖励。 |
| 认知水平 | 达到 70 | 银行定期存款收益从 10% 提高到 12%。 |
| 认知水平 | 达到 85 | 每完成一次储蓄目标，额外获得 1 点理财值。 |

### 4. 精力恢复与连续兼职限制

每名玩家自己的回合结束时，若本回合没有执行兼职或学习，自动恢复 10 点精力值。

玩家停留在宿舍地块时，可选择“休息”，恢复 25 点精力值和 15 点心情值。

玩家不能连续两个自己的回合执行兼职。系统需要记录 `lastActionType` 或 `lastWorkedTurn`，当上一回合动作为兼职时，本回合兼职按钮不可用。

### 5. 数值结局称号

结算时新增以下称号：

| 称号 | 获得条件 |
| --- | --- |
| 情绪管理大师 | 心情值最高且未进入过“低落”状态。 |
| 时间管理大师 | 精力值平均保持最高。 |
| 学霸担当 | 成绩最高。 |
| 认知升级王 | 认知水平最高。 |

## 五、社交与宿舍互动规则

### 1. 社交值

每名玩家拥有独立的社交值。社交值不直接等于资金，但会影响部分事件和结局称号。

获得社交值的方式：

- 在社团广场邀请玩家互动，双方各 +1。
- 参与团建，所有参与者各 +1。
- 帮助其他玩家支付费用，帮助者 +1。
- 在宿舍议事厅达成共同基金或共同储蓄计划，所有参与者各 +1。

社交值用途：

- 社交值达到 3 时，可抵消一次“强制出局类社交惩罚”。
- 社交值达到 5 时，可在一次收款类行动牌中额外获得 50 元人气收益。
- 结局时用于评定“宿舍核心”。

### 2. 团建参与机制

原规则中“累计 3 回合未参与任何团建的玩家强制出局”惩罚过重，容易造成负反馈。

改进后规则：

- 玩家每 3 回合至少应参与 1 次社交活动，包括团建、社团广场互动、宿舍议事厅协商或帮助室友。
- 若连续 3 回合未参与任何社交活动，获得“孤立”标记。
- 拥有“孤立”标记时，下一次获得行动牌数量 -1。
- 若再次连续 3 回合未参与社交活动，需扣除 150 元情绪消耗费。
- 社交值达到 3 的玩家可清除一次“孤立”标记。

该设计保留社交压力，但避免玩家因一次机制连锁直接失去游戏体验。

### 3. 室友互助

当玩家因被动事件或地块支出导致资金不足时，可向其他玩家发起一次互助请求。

其他玩家可选择：

- 无偿帮助：支付最多 100 元，帮助者获得 1 点社交值。
- 借款帮助：支付最多 200 元，受助者在 3 回合内归还本金，无利息。
- 拒绝帮助：无效果。

每名玩家每 4 回合最多发起 1 次互助请求。

## 六、主动行动牌规则

### 1. 抽取与使用规则

每回合固定抽取 1 张行动牌，手牌上限为 3 张。每回合仅可使用 1 张行动牌，使用后立即作废。

行动牌只产生金额增减、状态变化或社交互动，不产生重复叠加效果。

### 2. 行动牌效果清单

| 编号 | 行动牌 | 效果 |
| --- | --- | --- |
| 1 | 发起团建聚会 | 使用者选择是否邀请所有玩家参加。参加者各扣除 150 元并获得 1 点社交值；使用者不扣费但获得 1 点社交值。拒绝者无资金变化，但记录 1 次未参与社交。 |
| 2 | 举办生日会 | 向其余所有玩家各收取 150 元，全额计入使用者资金。所有支付者各获得 1 点社交值。 |
| 3 | 代取快递跑腿 | 其余每位玩家支付 80 元跑腿费，全部归使用者所有。使用者获得 1 点社交值。 |
| 4 | 拼单省钱 | 下回合自身所有个人消费支出减半。 |
| 5 | 宿舍分摊采购 | 所有玩家各扣除 100 元。若共同基金已建立，则可由共同基金优先支付。 |
| 6 | 申请助学金 | 自身立即获得 300 元。若当前资金最低，额外获得 50 元。 |
| 7 | 请客喝奶茶 | 自身扣除 200 元，其余每位玩家获得 50 元。使用者获得 1 点社交值。 |
| 8 | 兼职加班 | 自身额外获得 400 元，精力值 -35，心情值 -5，本回合无法进行储蓄操作。若上一回合已经兼职，则不能使用。 |
| 9 | 集体囤货 | 所有玩家各扣除 120 元，统一计入公共储备金。公共储备金可抵扣一次集体消费。 |
| 10 | 学霸辅导 | 向 1 名指定玩家收取 200 元辅导费，对方直接扣除对应金额。双方各获得 1 次学习互动记录；使用者认知水平 +3，接受辅导者成绩 +5。 |
| 11 | 闲置物品转卖 | 自身立即获得 250 元。 |
| 12 | 宿舍公摊缴费 | 自身扣除 300 元，其余所有玩家各返还 50 元。使用者获得 1 点社交值。 |
| 13 | 禁止 | 指定 1 名其他玩家本回合不能使用行动牌。若对方已经打出行动牌，则该行动牌作废。 |
| 14 | 制定预算 | 自身获得“预算”状态，接下来 2 回合个人消费支出 -50%，认知水平 +5。 |
| 15 | 室友饭搭子 | 指定 1 名玩家，本回合双方食堂或外卖消费各减少 40 元，双方各获得 1 点社交值和 5 点心情值。 |
| 16 | 二手教材互换 | 自身获得 150 元，指定 1 名玩家下次教学楼事件收益 +100 元，双方认知水平各 +2。 |

## 七、被动触发事件规则

### 1. 触发条件

玩家投掷骰子并移动落地后，随机触发 1 个被动事件。事件不可规避、不可拒绝。

### 2. 被动事件效果清单

| 编号 | 事件 | 效果 |
| --- | --- | --- |
| 1 | 捡到饭卡 | 随机获得 80-200 元。 |
| 2 | 快递丢失 | 扣除 150 元快递损失。 |
| 3 | 考试挂科补考 | 扣除 200 元补考费，成绩 -8，心情值 -15，精力值 -10，接下来 3 回合禁止参与任何兼职。 |
| 4 | 食堂饭菜免单 | 本回合无需支付食堂消费。 |
| 5 | 校园随机抽奖 | 获得 100 元幸运奖金。 |
| 6 | 水电费超支 | 额外扣除 100 元宿舍公摊费用。 |
| 7 | 捡到校园零钱 | 获得 50 元小额收益。 |
| 8 | 网购踩雷退货 | 扣除 100 元退货运费。 |
| 9 | 图书馆占座被罚 | 扣除 50 元罚款。 |
| 10 | 兼职额外奖励 | 若本回合执行兼职，则兼职收益翻倍；若未兼职，则无效果。 |
| 11 | 饭卡遗失补办 | 扣除 30 元补办费。 |
| 12 | 奖学金附加奖励 | 若本回合触发奖学金，则额外获得 200 元；否则获得 50 元学习鼓励金。成绩 +3，认知水平 +3。 |
| 13 | 室友帮忙代付 | 本次触发的支出可由 1 名指定室友承担。若室友同意，室友获得 1 点社交值。 |
| 14 | 校园消费券 | 本回合所有个人消费共减免 80 元。 |
| 15 | 社团招新 | 可选择扣除 50 元参加社团，获得 2 点社交值，心情值 +10，精力值 -10。 |

| 17 | 预算执行成功 | 若玩家本回合没有发生非必要消费，获得 100 元奖励、1 点理财值和 3 点认知水平。 |
| 18 | 宿舍夜谈 | 所有玩家可公开 1 张手牌；公开者获得 1 点社交值和 8 点心情值。 |

## 八、地块专属规则

| 网购点 | 消费 150 元；高风险高消费地块。 |
| 电影院 | 消费 120 元；若两名以上玩家同回合到达，可各获得 1 点社交值。 |
| 兼职中心 | 可执行兼职，获得 100-300 元，精力值 -30，心情值 -5。不能连续两个自己的回合兼职。 |
| 宿舍 | 每 2 回合结算水电费；可触发宿舍议事；可选择休息，恢复 25 点精力值和 15 点心情值。 |
| 银行 | 可办理定期存款、查看储蓄目标进度。认知水平达到 70 时，定期收益提高到 12%。 |
| 考点 | 扣除 100 元报名费；可能触发补考或学习奖励。 |
| 社团广场 | 可邀请 1 名玩家互动，双方各获得 1 点社交值，心情值 +10，精力值 -10。 |
| 心愿墙 | 可公开个人目标，完成后获得 100 元或 1 点理财值。 |
| 宿舍议事厅 | 所有玩家可协商共同基金、共同储蓄或互助方案。 |

## 九、状态标记

| 状态 | 来源 | 效果 |
| --- | --- | --- |
| 补考中 | 考试挂科补考 | 3 回合内禁止兼职。 |
| 预算 | 制定预算 | 接下来 2 回合个人消费支出 -50%。 |
| 孤立 | 连续 3 回合未参与社交 | 下一次获得行动牌数量 -1。 |
| 学习状态 | 图书馆、教学楼 | 提高奖学金触发概率或减免考试相关费用。 |
| 消费券 | 校园消费券 | 本回合个人消费共减免 80 元。 |
| 拼单省钱 | 行动牌 | 下回合个人消费支出减半。 |
| 低落 | 心情值低于 30 | 不能主动发起团建，消费类事件额外扣除 50 元。 |
| 精力不足 | 精力值低于 30 | 不能兼职，不能学习。 |
| 心情良好 | 心情值达到 85 | 下一次社交互动额外获得 1 点社交值。 |

## 十、推荐数值平衡

为保证游戏既有竞争性，也能体现储蓄与社交价值，建议采用以下平衡原则：

1. 单次攻击类行动牌的损失不宜超过初始生活费的 15%。
2. 单次收入类行动牌的收益不宜超过初始生活费的 20%。
3. 社交收益主要体现为减免、互助、结局称号，不应完全替代资金胜利。
4. 储蓄收益要稳定但不能过高，避免玩家只存钱不互动。
5. 破产机制保留紧张感，但应允许玩家通过互助或共同基金获得一次缓冲。
6. 心情值、精力值、成绩、认知水平应主要影响行动选择和长期收益，不应直接替代资金胜负。
7. 兼职收益要受精力与连续兼职限制约束，避免玩家只通过重复兼职获胜。

## 十一、设计目标对应表

| 目标问题 | 对应改进机制 | 预期效果 |
| --- | --- | --- |
| 宿舍交流弱 | 宿舍议事厅、宿舍共同基金、室友互助、宿舍夜谈 | 促使玩家开口协商、表达需求、讨论策略。 |
| 储蓄思维不足 | 银行定期存款、储蓄目标、预算状态、理财值 | 让玩家体验延迟消费和计划资金的收益。 |
| 校园社交渠道不足 | 社团广场、团建、饭搭子、生日会、社团招新 | 将校园社交转化为可玩的资源和事件。 |
| 过度互相伤害 | 社交值、共赢结局、共同基金、互助请求 | 让合作也能带来优势，降低纯攻击体验。 |
| 随机性过强 | 心愿墙目标、预算牌、定期存款、共同基金 | 给玩家更多可规划策略。 |
| 作息与学习失衡 | 心情值、精力值、成绩、认知水平、连续兼职限制 | 让玩家在赚钱、学习、休息和社交之间做取舍。 |

## 十二、推荐对局流程

1. 所有玩家选择初始生活费。
2. 每名玩家设置一个储蓄目标。
3. 初始化每名玩家的心情值、精力值、成绩、认知水平。
4. 洗混行动牌和被动事件牌。
5. 玩家轮流投骰、移动、结算地块和事件。
6. 每 4 回合进行一次父母转账。
7. 每 5 回合进入一次宿舍小结，玩家可公开资金、储蓄目标或社交计划。
8. 游戏持续到只剩 1 名玩家，或达到 20 回合后触发结局评定。

### 8. 后端接口建议

| 接口 | 用途 |
| --- | --- |
| `POST /api/rooms` | 创建宿舍房间。 |
| `POST /api/rooms/:roomId/players` | 加入玩家。 |
| `POST /api/rooms/:roomId/start` | 初始化游戏状态。 |
| `GET /api/games/:gameId` | 获取完整 `GameState`。 |
| `POST /api/games/:gameId/roll` | 投骰、移动并触发地块与被动事件。 |
| `POST /api/games/:gameId/tile-action` | 执行兼职、学习、休息、储蓄、社交等地块行动。 |
| `POST /api/games/:gameId/use-card` | 使用行动牌。 |
| `POST /api/games/:gameId/end-turn` | 结束回合，结算收益、恢复和状态倒计时。 |

关键约定：所有接口返回更新后的完整 `GameState` 和本次操作的 `TurnResult`。前端不得自行计算兼职限制、数值变化、破产和胜负，只负责展示后端返回结果。

## 十四、版本说明

本规则是在原有校园版大富翁规则基础上进行的目标导向改进版。改进重点不是单纯增加事件数量，而是让玩家在资金竞争中同时面对社交选择、储蓄规划和宿舍协商，从而更贴近大学宿舍生活场景。
